# Ashiva Manifests
Ashiva uses **Manifest Files**  to store page-level and site-level data in `JSON` format:

- `pageManifest`
- `siteManifest`

Either **Manifest** may be populated manually, via a GUI, computationally, or via data from its associated **ManifestData** file:

- `pageManifestData`
- `siteManifestData`
